# User_Login_Spring_Java
 # BackEnd Login Registration
 
 This is a complete Login backend system using Spring Boot.
 
	✔️ Java (BackEnd)
  ✔️ SpringBoot \
  ✔️ Spring Security \
  ✔️ E-Mail Verification w/ Expiry \
  ✔️ HTML \
  ✔️ PostgresSQL \
  ✔️ MailDEv (SMTP Server used for generating email verification)
